The anscombe folder visualizes Anscombe´s Quartet.

The following files are used for to create the Raincloud Quartet:
helperFunctions.R
simulateSeedData.R
quartetData folder (.jasp file in here is to calculate confidence interval for Cohen´s d)

The evaluation of plots from the Psychonomic Bulletin & Review (PBR) needs:
dataPbrPlotting.xlsx
analyseDataPbrPlotting.R

The data to showcase raincloud plots in JASP needs:
createShowcaseData.R
showcaseData folder
